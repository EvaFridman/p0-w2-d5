# p0-w2-d5
Необходимо создать сайт по макету лендинга на чистых HTML и CSS.

План релизов:
1. Структура HTML (index.html).
2. Нормализация (normalize.css).
3. Переменные (variables.css).
4. Кнопки, переключатели (ui-kit.css).
5. Основные стили для mobile (styles.css).
6. Стили для планшета и десктопа.
7. Финальная версия.
